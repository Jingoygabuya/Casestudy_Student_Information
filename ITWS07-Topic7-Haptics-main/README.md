# Student Management System - Basic CRUD

## Quick Setup

### 1. Database (XAMPP)
```
1. Start XAMPP (Apache + MySQL)
2. Go to: http://localhost/phpmyadmin
3. Import: backend/database/student_db.sql
```

### 2. Backend
```
Copy 'backend' folder to: C:\xampp\htdocs\student-management\
```

### 3. Run App
```powershell
$env:Path = "C:\Program Files\nodejs;" + $env:Path
cd C:\Users\Admin\Desktop\ITWS07-Topic7-Haptics-main
ionic serve
```
